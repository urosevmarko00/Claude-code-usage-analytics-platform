## Tools Used

ChatGPT (GPT-4)

## Use Cases

The LLM was used as a development assistant for:

+ Discussing and refining project architecture ideas
+ Generating parsing logic for JSON telemetry logs
+ Suggesting improvements for the analytics pipeline
+ Suggesting Streamlit dashboard layouts
+ Performing occasional code review and refactoring suggestions
+ Generating documentation (README)
